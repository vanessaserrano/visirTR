This is the home for the VISIR/HIVE Dashboard (VISIR/HIVE-DB) project. It applies learning analytics approaches to the VISIR & HIVE remote labs logged data.

It is currently implemented as a Shiny app in R.

Current release (version 20250827)

OPTION 1. Standalone VISIR/HIVE-DB (Windows OS only)

This file includes a portable version of R and VISIR/HIVE-DB. It runs as a standalone application.

To run the dashboard:

  1. Download the current portable version: VISIR_DB_20250827.zip.
  2. Unzip the file.
  3. Launch VISIR_DB or VISIR_DB.dat. The dashboard may take some seconds to be fully available.
  4. Enjoy!

OPTION 2. Running VISIR/HIVE-DB from R (any OS with R installed)

R is required to use VISIR/HIVE-DB. To install R, follow the instructions from the R project.

  1. Download the R script to download and run the current version of VISIR-DB: runVISIRDB.R.
  2. Source (execute) this script in your R interpreter.
  3. Enjoy!

Development version

The development files for VISIR/HIVE-DB are available at the following GitHub project: https://github.com/vanessaserrano/visirTR. The development version can be run from the script runVISIRDB_dev.R.
